# Funkcss

## Installation

Add this line to your application's Gemfile:

```ruby
gem "funkcss", github: 'filipelinhares/funkcss-gem', branch: :master
```

And then execute:
```sh
bundle
```

## Usage
`application.css`
```scss
@import 'funk-components';
// or
@import 'funk';
```
